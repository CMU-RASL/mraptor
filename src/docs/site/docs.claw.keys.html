Claw's Keyboard Shortcut Reference

We have made a concerted effort to provide reasonably intuitive key bindings for as many Claw commands as possible. All of Claw's functionality is available through the main menu bar, and key bindings are displayed within the menus as a reminder. The rather unintuitive bindings of some of these options are forced by the hardcoded bindings in the entry widget (used to provide input to processes' stdin), since we avoid conflicts so as to make the keybindings consistent throughout the program. Note that much of this functionality is also available in the context menus (i.e. right-click menus).
Shortcut Description
Ctrl+R Run currently selected process(es) [*]
Ctrl+T Terminate (kill) currently selected process(es) [*]
Ctrl+S Show (view) currently selected process(es) [*]
Ctrl+M Mask (hide) currently selected process(es) [*]
Ctrl+O Switch to Overview tab (shows all currently viewed processes' output)
Ctrl+Y Update status of currently selected process(es) [*]. Note that this will have little to no effect in most cases; Claw now automatically updates process state every 30 seconds.
For the above shortcuts, Shift+Ctrl+key performs the given action for all known processes, rather than the currently selected ones.
Ctrl+Tab Switch to the next tab in the process interaction area (note that this will change the currently selected process, unless the tie seletion to tab Claw preference has been changed to '0').
Ctrl+Shift+Tab Switch to the previous tab in the process interaction area (note that this will change the currently selected process, unless the tie seletion to tab Claw preference has been changed to '0').
Ctrl+L Listen (subscribe to) currently selected process(es) output [*]
Ctrl+N u(N)subscribe from currently selected process(es) output [*]
Alt+M Clear the current process' output buffer (this is equivalent to clear in an xterm).
Ctrl+G Set process confi(G)uration for currently selected process [*]
Ctrl+I Load configuration file stored on cl(I)ent machine
Ctrl+J Reload most recently loaded client-local config file. This reloads from disk, and will thus pick up any changes.
Shft+Ctrl+I Load a configuration file stored on a daemon machine
Ctrl+Z Swallow an arbitrary X window into a tab (after selecting this, your mouse cursor will change; the next window you click on will be swallowed). You may later unswallow it; this provides a simple way to avoid screen clutter and bring all system components together.
Ctrl+P Open the claw preferences dialog
Ctrl+Q Quit the client
Within the entry line for a given process, most Emacs-ish keybindings hold (since they're hard-coded into the GTK widget, they force some of the (rather non-intuitive) bindings above):
Ctrl+A Go to beginning of line
Ctrl+B Move back one character
Ctrl+C Copy to clipboard
Ctrl+D Delete forward one character
Ctrl+E Move to end of line
Ctrl+F Move forward one character
Ctrl+H Delete backward one character
Ctrl+K Delete from cursor to end of line
Ctrl+U Delete line
Ctrl+V Paste from clipboard
Ctrl+W Delete backward one word
Ctrl+X Cut to clipboard
Alt+B Move backward one word
Alt+D Delete forward one word
Alt+F Move forward one word
[*] Note: "currently selected process(es)" refers to the zero or more processes highlighted in the process tree widget in the upper-left pane of the GUI.

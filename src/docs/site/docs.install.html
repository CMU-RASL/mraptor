Installation Overview

As with anything, there are a number of ways to install Microraptor: RPM, SRPM, and build from source. We recommend them in precisely that order: if an RPM is available for your system, use the sucker. If not, try building a binary RPM from the SRPM (see instructions). If you're successful, please drop us a copy to save others the work. If RPMs just don't do it for you, build from the source using these instructions, although be warned: it can be a bit hairy. If you would like to contribute a package for your distribution, we would be eternally grateful.

Regardless, you need a working IPC installation before installing Microraptor. We provide unofficial RPMs of the latest version of IPC here. If you prefer to obtain it straight from the source, go to http://www.cs.cmu.edu/~ipc for the latest version of IPC. It's relatively easy to build, so don't worry about it too much. :-)

Once you've installed Microraptor, continue on to the configuration instructions.
